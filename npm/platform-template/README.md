# @panache-cli/__NAME__

This is an internal platform-specific package containing a prebuilt
[panache](https://panache.bz) binary for `__NAME__`.

You should not depend on it directly. Install
[`@panache-cli/panache`](https://www.npmjs.com/package/@panache-cli/panache)
instead --- it will pull in the right binary for your platform automatically:

```sh
npm install -g @panache-cli/panache
```
