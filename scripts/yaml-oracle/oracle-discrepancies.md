# Oracle discrepancies: pandoc_direct vs psych_libyaml

For frontmatter, **pandoc_direct is ground truth** --- pandoc wraps the YAML as
a metadata block and adds rules (e.g. metadata must be a mapping) on top of
libyaml. These rows need human classification in Phase 1.
