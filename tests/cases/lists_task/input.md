## Basic Task Lists

- [ ] Unchecked task
- [x] Checked task
- [ ] Another unchecked task
- [X] Checked with capital X

## Task Lists with Different Bullets

* [ ] Task with asterisk
* [x] Completed task

+ [ ] Task with plus
+ [x] Completed task

## Task Lists with Content

- [ ] Task with **bold** text
- [x] Completed task with `code`
- [ ] Task with [a link](https://example.com)

## Nested Task Lists

- [ ] Parent task
  - [ ] Nested unchecked task
  - [x] Nested checked task
- [x] Another parent task

## Mixed Regular and Task Lists

Regular list items:

- Regular item one
- Regular item two

Task list items:

- [ ] Task item one
- [x] Task item two

Back to regular:

- Regular item three
