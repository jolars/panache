# Code Spans

Simple code: `code`

With spaces: `code with spaces`

Multiple backticks: ``code with `backtick` inside``

Code in emphasis: *text with `code` inside*

Adjacent code: `first` and `second` code spans

Escaped backtick outside: \`not code\`
