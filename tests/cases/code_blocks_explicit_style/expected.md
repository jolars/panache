# Code Blocks with Explicit Style

Basic shortcut syntax:

```{.python}
print("hello")
```

Shortcut with attributes:

```{.python .numberLines}
x = 42
```

Already explicit:

```{.haskell #mycode}
qsort [] = []
```

Multiple classes:

```{.python .numberLines startFrom="10"}
def foo():
    pass
```
