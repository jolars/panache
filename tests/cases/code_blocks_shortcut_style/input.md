# Code Blocks with Shortcut Style

Explicit syntax with single class:

``` {.python}
print("hello")
```

Explicit with attributes:

``` {.python .numberLines startFrom="5"}
x = 42
```

Multiple classes only:

``` {.python .dark}
y = 10
```

Just attributes (no language):

``` {#myid key="value"}
plain text
```
